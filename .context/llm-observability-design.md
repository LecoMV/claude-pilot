# LLM Management, Observability & System Services Design

## Overview

Complete visibility and control over:
- Local LLM management (Ollama, llama.cpp, vLLM, etc.)
- pgvector for PostgreSQL embeddings
- Real-time agent/swarm/hive logging
- System services with watchdogs and auto-recovery
- Unified logging console for everything
